⭐ Guía de instalación ⭐


Para instalar la aplicación:

Descargar el archivo comprimido.

Descomprimir el archivo.


Para abrir la aplicación:

Abrir la carpeta descomprimida (LibraryManager).

Ejecutar "Library Manager.vbs".


Para cambiar los datos de acceso de la base de datos:

Abrir la carpeta LibraryManager/app.

Abrir el archivo "database.propierties".

Modificar los datos de acceso.

⭐ Requisitos ⭐


Para instalar y usar la aplicación se necesita:


Hardware:

Arquitectura: Sistema con arquitectura de 64 bits.

Procesador: Mínimo de 2 núcleos; se recomienda 4 núcleos o más para un rendimiento óptimo.

Memoria RAM: Al menos 4 GB; se recomienda 8 GB o más para una experiencia fluida.

Almacenamiento: Espacio disponible de al menos 500 MB para la instalación y funcionamiento de la aplicación.

Resolución de pantalla: Mínima de 1200x700 píxeles; se recomienda 1920x1080 píxeles para una visualización óptima de la interfaz gráfica.


Software:

Sistema Operativo: Windows 10 o Windows 11, ya sea como sistema operativo principal o ejecutado en una máquina virtual.

Java Development Kit (JDK): Versión 22 o superior, correctamente instalada y configurada en el sistema.

Herramienta de compresión/descompresión: Software como WinRAR, 7-Zip o equivalente, para la gestión de archivos comprimidos.

Navegador web: Cualquier navegador actualizado para acceder a funcionalidades web adicionales, si aplica.


Conectividad

Conexión a internet estable sin restricciones que impidan el acceso a páginas web.
